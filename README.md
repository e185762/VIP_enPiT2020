## エレベーターピッチ
`自分のイメージする洋服が探しづらい`を解決したい`洋服を探したい人`向けの

`簡単に洋服検索できる`を実現する`DFS`です。

これは`絵から洋服を検索すること`ができ、`既存の画像洋服検索`とは違って

`自分のイメージに近い洋服の画像を探さなくていいという利点`が備わっています。
